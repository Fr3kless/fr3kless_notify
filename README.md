## Fr3kless_Notify
- Simple FiveM Notification
## In Game Preview Command
- /testfr3klessnotify
## Triggers:
- TriggerEvent('fr3kless_notify:shownotify', title, content, timeout)
## Sample:
- TriggerEvent('fr3kless_notify:shownotify', 'Powiadomienie', 'realfr3kless#0', 6000)
